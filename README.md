# Taiyaki

Slack bot.

Ask it where a user is and it will notify you when the user comes online.

```
  mix deps.get
```

### Run it

Create a `.env` file in which you sent your generated Slack token. (you can use `dot_env` as a template)

```
source .env

mix run --no-halt

```

### Run tests

```
mix test
```
